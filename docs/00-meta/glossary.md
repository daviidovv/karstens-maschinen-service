# Glossar

Letzte Aktualisierung: 2026-02-19

## Fachbegriffe

### Kuvertiermaschine
Maschine zum automatischen Falten und Einlegen von Dokumenten in Umschläge.

### Adressdrucker
Gerät zum Bedrucken von Umschlägen und Etiketten mit Adressen und Grafiken.

### Frankiermaschine
Maschine zur automatischen Portoabrechnung und Freimachung von Postsendungen.

### Verbrauchsmaterial
Zubehör wie maschinenkompatible Umschläge, Siegelflüssigkeit, Tintenpatronen und Wartungsmaterialien.

### B2B (Business-to-Business)
Geschäftsbeziehung zwischen Unternehmen, nicht mit Endverbrauchern.

### Gebrauchtmaschine
Bereits genutzte Postbearbeitungsmaschine, die angekauft oder verkauft wird.

## Produktkategorien

### Falz- und Kuvertiermaschinen
z.B. Pitney Bowes Relay 1000, Neopost DS Serie

### Software-Lösungen
z.B. Flexmail für dynamische Druckprozesse

## Abkürzungen

### Technisch
- **API** - Application Programming Interface
- **DTO** - Data Transfer Object
- **NFR** - Non-Functional Requirement
- **ADR** - Architecture Decision Record
- **MVP** - Minimum Viable Product
- **CRUD** - Create, Read, Update, Delete

### Projekt-spezifisch
- **DoD** - Definition of Done
- **REQ** - Requirement (Anforderung)
- **US** - User Story
