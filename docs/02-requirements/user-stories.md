# User Stories

Letzte Aktualisierung: 2026-02-19

## Format

```
Als [Rolle]
möchte ich [Aktion]
damit [Nutzen]
```

## Kunde (Geschäftskunde)

### US-001: Produkte durchsuchen
Als Geschäftskunde
möchte ich alle verfügbaren Postbearbeitungsmaschinen sehen
damit ich das passende Produkt für mein Unternehmen finden kann.

### US-002: Produkte filtern
Als Geschäftskunde
möchte ich Produkte nach Kategorie filtern können
damit ich schneller finde was ich suche.

### US-003: Anfrage stellen
Als Geschäftskunde
möchte ich eine Anfrage für ein Produkt stellen können
damit ich ein individuelles Angebot erhalte.

### US-004: Gebrauchtmaschinen ansehen
Als Geschäftskunde
möchte ich verfügbare Gebrauchtmaschinen sehen
damit ich Kosten sparen kann.

### US-005: Maschine zum Verkauf anbieten
Als Geschäftskunde
möchte ich meine alte Maschine zum Verkauf anbieten können
damit ich sie loswerden kann.

## Administrator

### US-010: Produkte verwalten
Als Administrator
möchte ich Produkte anlegen und bearbeiten können
damit der Katalog aktuell bleibt.

### US-011: Anfragen bearbeiten
Als Administrator
möchte ich alle Kundenanfragen sehen und bearbeiten können
damit keine Anfrage verloren geht.

### US-012: Kategorien verwalten
Als Administrator
möchte ich Kategorien anlegen und bearbeiten können
damit die Produkte strukturiert sind.
