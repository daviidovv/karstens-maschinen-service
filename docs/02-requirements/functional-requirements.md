# Funktionale Anforderungen

Letzte Aktualisierung: 2026-02-19

## Status

**Grob definiert** - Wird in Task "Vision ausarbeiten" detailliert.

## Produktkatalog

### REQ-001: Produkte anzeigen
- Benutzer kann alle Produkte sehen
- Produkte werden mit Bild, Name, Beschreibung angezeigt

### REQ-002: Produkte filtern
- Benutzer kann nach Kategorie filtern
- Benutzer kann nach Suchbegriff suchen

### REQ-003: Produktdetails
- Benutzer kann Detailseite eines Produkts aufrufen
- Technische Spezifikationen werden angezeigt

## Anfragen

### REQ-010: Produktanfrage
- Benutzer kann Anfrage für ein Produkt stellen
- Pflichtfelder: Firmenname, Ansprechpartner, E-Mail, Nachricht
- Optional: Telefon

### REQ-011: Allgemeine Kontaktanfrage
- Benutzer kann allgemeine Anfrage stellen
- Kontaktformular auf eigener Seite

## Gebrauchtmaschinen

### REQ-020: Gebrauchtmaschinen anzeigen
- Eigene Sektion für Gebrauchtmaschinen
- Mit Zustand und Baujahr

### REQ-021: Gebrauchtmaschinen-Ankauf
- Formular für Verkaufsanfragen
- Felder: Maschinentyp, Baujahr, Zustand, Kontaktdaten

## Administration

### REQ-030: Produkte verwalten
- Admin kann Produkte anlegen, bearbeiten, löschen
- Admin kann Kategorien verwalten

### REQ-031: Anfragen verwalten
- Admin sieht alle Anfragen
- Admin kann Status ändern (Neu, In Bearbeitung, Erledigt)

### REQ-032: Login
- Admin muss sich einloggen
- Geschützter Bereich
