## Ziel

Die vision.md mit den konkreten Anforderungen für "Karstens Maschinen" füllen - eine Website für den Verkauf von reparierten Postbearbeitungsmaschinen (B2B).

## Kontext

**Geschäftsmodell des Bekannten:**
- Kauft kaputte Postbearbeitungsmaschinen an
- Repariert sie
- Verkauft sie wieder (aktuell über eBay)
- Möchte eigene professionelle Website wie aufpost.de

**Warum eigene Website:**
- Professionellerer Auftritt als eBay
- Größeres Sortiment zeigen (auch Maschinen die auf Anfrage beschaffbar sind)
- Unabhängigkeit von eBay-Gebühren

## Geklärte Anforderungen

### Firmenname
- **Karstens Maschinen**

### Preismodell
- Preise werden angezeigt
- Kein direkter Online-Kauf (kein Warenkorb)
- Kauf nur auf Anfrage (Angebot wird erstellt)

### Verfügbarkeit
- **Auf Lager**: Maschinen die aktuell da sind
- **Auf Anfrage**: Größeres Sortiment das beschafft werden kann

### Ankauf-Bereich
- Ja, Besucher können eigene Maschinen zum Verkauf anbieten
- Formular mit: Maschinentyp, Zustand, Baujahr, Kontaktdaten, Fotos

### Kontaktmöglichkeiten
- Anfrage-Button direkt bei jedem Produkt
- Allgemeines Kontaktformular

### Admin-Bereich
- Der Bekannte soll selbst Maschinen eintragen/bearbeiten können
- Kann etwas komplexer sein (technisch versiert genug)
- Funktionen:
  - Maschinen hinzufügen/bearbeiten/löschen
  - Fotos hochladen
  - Preise setzen
  - Status ändern (Auf Lager / Auf Anfrage / Verkauft)
  - Kategorien verwalten
  - Anfragen einsehen

### Produktkategorien
- **NOCH ZU KLÄREN** mit dem Bekannten
- Vorschlag (wie aufpost.de):
  - Kuvertiermaschinen
  - Frankiermaschinen
  - Adressdrucker
  - Zubehör/Verbrauchsmaterial
  - (oder eigene Kategorien)
